# StatusBar

## Props

Inherits all props from the React Native StatusBar except translucent, which is automatically set to `true`.

### `backgroundColor`

| type     | default       | required |
| -------- | ------------- | -------- |
| `string` | `transparent` | NO       |

### `height`

| type     | default                       | required |
| -------- | ----------------------------- | -------- |
| `number` | `constants.STATUS_BAR_HEIGHT` | NO       |
